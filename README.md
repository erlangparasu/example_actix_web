# example_actix_web

```shell
git clone https://github.com/erlangparasu/example_actix_web.git
```

## How To Run

```shell
cd example_actix_web
```

```shell
cargo run --release
```

```shell
curl -X GET http://127.0.0.1:8181/
```

## Run via Docker

```shell
docker compose build
```

```shell
docker compose up -d
```

```shell
curl -X GET http://127.0.0.1:8181/
```
